# ROB-346: Training Curriculum for Competition Robots (VEX/FLL 2026)

## Overview
This curriculum prepares students aged 10-16 for VEX and FLL robotics competitions over an 8-week intensive program.

---

## Week 1-2: Robot Building Fundamentals

### Week 1: Mechanical Foundations
| Day | Topic | Activities |
|-----|-------|------------|
| 1 | Parts identification | Sort and identify VEX/FLL kit components |
| 2 | Basic structures | Build frames, mounts, and connectors |
| 3 | Drive systems | Construct drivetrains (tank, mecanum, swerve) |
| 4 | Transmissions | Gears, chains, and gear ratios |
| 5 | Assessment | Build a basic driving base in 45 min |

### Week 2: Advanced Mechanisms
| Day | Topic | Activities |
|-----|-------|------------|
| 1 | Arm systems | Lift arms, linear slides |
| 2 | Grabbing mechanisms | Claws, intakes, conveyors |
| 3 | Sensor integration | Motors, encoders, gyros |
| 4 | Build challenge | Design a ball collector mechanism |
| 5 | Assessment | Build a functional grab-and-lift prototype |

---

## Week 3-4: Programming Fundamentals for Competition

### Week 3: Core Programming
| Day | Topic | Activities |
|-----|-------|------------|
| 1 | Blockly for competition | Commands, loops, conditionals |
| 2 | Motor control | Speed, torque, precision positioning |
| 3 | Sensor programming | Touch, light, ultrasonic, gyro |
| 4 | Autonomous routines | Path planning and execution |
| 5 | Assessment | Program autonomous "Maze Runner" |

### Week 4: Advanced Programming
| Day | Topic | Activities |
|-----|-------|------------|
| 1 | PID control | Line following, precise turns |
| 2 | Multi-sensor fusion | Combine inputs for accuracy |
| 3 | State machines | Competition mode management |
| 4 | Debugging | Live monitoring, error correction |
| 5 | Assessment | Program a full autonomous challenge |

---

## Week 5-6: Competition Strategy & Game Analysis

### Week 5: Game Analysis
| Day | Topic | Activities |
|-----|-------|------------|
| 1 | VEX 2026 rules | Field setup, scoring, restrictions |
| 2 | FLL 2026 theme | Mission breakdown, point values |
| 3 | Strategy formation | Tiers: high-value vs. consistent |
| 4 | Time management | 2.5 min autonomous + 1.5 min driver |
| 5 | Assessment | Present 3-game strategy plan |

### Week 6: Opponent Analysis
| Day | Topic | Activities |
|-----|-------|------------|
| 1 | Scout other teams | Watch match videos |
| 2 | Counter-strategies | Defensive and offensive plays |
| 3 | Alliance selection | Partner strengths/weaknesses |
| 4 | Match planning | Role assignment (driver, coach, scout) |
| 5 | Assessment | Run mock alliance selection |

---

## Week 7: Practice Competition Simulations

### Weekly Schedule
| Day | Activity |
|-----|----------|
| Monday | Dry run: Full autonomous routines |
| Tuesday | Driver practice: Skill challenges |
| Wednesday | Mock matches: 2 vs 2 alliances |
| Thursday | Mock matches: Practice tournament |
| Friday | Review & iterate: Fix weaknesses |

### Simulation Rules
- Full competition environment (field, robots, timer)
- Judges present for feedback
- Video recording for review
- Parents invited to final showcase

---

## Week 8: Skills Assessment & Final Prep

### Skills Assessment Rubric

#### 1. Robot Build (25 points)
| Criteria | 5 (Excellent) | 3 (Adequate) | 1 (Needs Work) |
|----------|---------------|--------------|-----------------|
| Structural integrity | Rigid, no wobble | Minor flex | Significant flex |
| Mechanism efficiency | Smooth, fast | Functional | Slow/jams |
| Cable management | Clean, protected | Basic routing | Tangled |
| Weight optimization | Within limit, balanced | Slightly over | Over limit |

#### 2. Programming (25 points)
| Criteria | 5 (Excellent) | 3 (Adequate) | 1 (Needs Work) |
|----------|---------------|--------------|-----------------|
| Autonomous reliability | 90%+ success | 70-89% | <70% |
| Code efficiency | Clean, documented | Basic comments | Messy/spaghetti |
| Sensor accuracy | Consistent readings | Occasional errors | Frequent errors |
| Innovation | Unique solutions | Standard approach | Copied/unmodified |

#### 3. Competition Performance (25 points)
| Criteria | 5 (Excellent) | 3 (Adequate) | 1 (Needs Work) |
|----------|---------------|--------------|-----------------|
| Driver skill | Smooth, precise | Basic control | Erratic |
| Strategy execution | Matches plan | Partial | No clear plan |
| Adaptability | Adjusts mid-match | Slow adaptation | Frozen |
| Team coordination | Excellent | Basic | Poor |

#### 4. Game Knowledge (25 points)
| Criteria | 5 (Excellent) | 3 (Adequate) | 1 (Needs Work) |
|----------|---------------|--------------|-----------------|
| Rule knowledge | Expert level | Basic understanding | Unfamiliar |
| Scoring optimization | Maximizes points | Average scoring | Low scoring |
| Safety awareness | Always safe | Usually safe | Unsafe behavior |
| Sportsmanship | Exemplary | Acceptable | Poor |

## Competition Day Checklist

### Pre-Competition
- [ ] Robot inspection passed
- [ ] Spare batteries charged (8+)
- [ ] Tools kit complete
- [ ] Backup code saved
- [ ] Match schedule confirmed
- [ ] Alliance partner contact info

### During Competition
- [ ] Arrive 30 min before first match
- [ ] Check field for calibration
- [ ] Scout opposing teams
- [ ] Brief drivers on strategy
- [ ] 5-min autonomous practice
- [ ] Final robot check before each match

### Post-Competition
- [ ] Cool-down debrief
- [ ] Video/photo documentation
- [ ] Awards ceremony attendance
- [ ] Thank volunteers/judges
- [ ] Begin off-season planning

## Required Equipment per Team

### Robot Kit
- Competition chassis (custom or VEX/FLL kit)
- 4+ motors with encoders
- Gyroscope sensor
- 2x ultrasonic sensors
- Color/light sensor
- Touch sensor
- Battery (VEX) / Core Set (FLL)

### Tools
- Screwdrivers (Phillips, flathead)
- Hex keys (metric)
- Pliers (needle-nose, regular)
- Zip ties (various sizes)
- Electrical tape
- Lubricant (for gears)
- Measuring tape

### Documentation
- Engineering notebook
- Team portfolio
- Robot specification sheet
- Code backup (USB + cloud)

---

## Mentorship Requirements

Each competition team needs:
- 1 Head Mentor (technical lead)
- 1 Driver (student)
- 1 Co-pilot/strategy (student)
- 1 Scout (student)
- 1 Builder (student)
- Optional: Parent chaperone
