# Lesson Design Skill

## Overview
Creating engaging Vietnamese educational content for kids learning robotics.

## Lesson Structure
```markdown
# [Bài N]: Tên Bài Học

## 🎯 Mục Tiêu
- Mục tiêu 1
- Mục tiêu 2

## 📚 Giới Thiệu (5 phút)
Nội dung giới thiệu bằng ngôn ngữ trẻ em

## 👨‍💻 Thực Hành (15 phút)
Hướng dẫn từng bước

## 🎮 Thử Thách (10 phút)
Project nhỏ để áp dụng

## ❓ Câu Hỏi
3-5 câu hỏi trắc nghiệm

## 🏆 Đánh Giá
- Hoàn thành: ✓/✗
- Điểm mạnh:
- Cần cải thiện:
```

## Age Groups
| Age | Complexity | Focus |
|-----|------------|-------|
| 6-8 | Basic | Movement, simple commands |
| 8-10 | Intermediate | Sensors, loops |
| 10-12 | Advanced | AI basics, complex projects |
| 12-16 | Expert | Competition-level |

## Curriculum Tracks
1. **Beginner Track** (6-10 tuổi)
   - Hello Robot
   - Movement basics
   - Simple sequences
   - Maze navigation

2. **Intermediate Track** (10-14 tuổi)
   - Sensor programming
   - Line following
   - AI introduction
   - Mini competitions

3. **Advanced Track** (14-16 tuổi)
   - VEX/FLL competition prep
   - Complex autonomous programs
   - Team projects

## Writing Tips
1. Use simple Vietnamese
2. Add emojis for fun
3. Include real photos/videos
4. Add relatable examples
5. Celebrate small wins

## Quiz Format
```markdown
## ❓ Câu Hỏi

**Câu 1:** Robot cần gì để di chuyển?

- [ ] A. Cánh quạt
- [ ] B. Động cơ và bánh xe ✓
- [ ] C. Pin năng lượng mặt trời
- [ ] D. Loa

**Giải thích:** Robot cần động cơ để tạo lực và bánh xe để di chuyển!
```
