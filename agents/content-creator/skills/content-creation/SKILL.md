# Content Creation Skill

## Overview
Creating engaging content for parents and kids on social media.

## Platform Strategy
| Platform | Target | Content Type | Frequency |
|----------|--------|--------------|-----------|
| Facebook | Parents | Educational posts, testimonials | 3x/week |
| YouTube | Parents, kids | Tutorial videos, demos | 1x/week |
| TikTok | Kids | Fun robot clips, challenges | 3x/week |
| Instagram | Mixed | Photos, stories | 2x/week |

## Content Pillars
1. **Education** - Show learning happening
2. **Fun** - Kids enjoying robotics
3. **Results** - Projects completed
4. **Behind scenes** - How robots are made

### TikTok Video Script
```
Script: Robot Dance Challenge

[0-3s] Kids unbox robot kit - excitement!
[3-8s] Build the robot (time-lapse)
[8-15s] Program dance moves in Blockly
[15-20s] Robot dances, kids cheer
[20-25s] Call to action - "Duet this dance!"
```

## Image Guidelines
- Bright, colorful images
- Kids actively engaged
- Robots clearly visible
- Natural expressions
- Safe environments

## Video Guidelines
- Vertical (9:16) for TikTok/Reels
- Horizontal (16:9) for YouTube
- Under 60 seconds for social
- Hook in first 3 seconds
- Clear audio
